## Images Needing Attention

### Skip These (wall as background, not subject)
- Gallery/showroom shots where plaster is incidental
- Poorly lit or blurry images
- Images with distracting clutter

### Need More Of
- Before/after pairs showing substrate → finished
- Daragh at work (craftsman shots)
- Material mixing/preparation
- Tool closeups

## Suggested Folder Structure

```
public/images/
├── hero/           # Best 3-5 wide room shots
├── authority/      # Corner details, transitions
├── process/        # Masking, scaffolding, prep
├── before-after/   # Transformation pairs
├── finishes/
│   ├── venetian/   # Polished, reflective
│   ├── limewash/   # Matte, chalky
│   ├── marmorino/  # Stone-like
│   ├── tadelakt/   # Waterproof, bathroom
│   └── microcement/ # Modern seamless
└── brand/          # Branded pieces, showroom
```
